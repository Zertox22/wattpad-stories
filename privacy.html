Politique de Confidentialité

Dernière mise à jour : Juin 2026

1. Introduction

La présente Politique de Confidentialité a pour but de vous expliquer de manière simple, transparente et claire comment sont traitées les données lors de votre navigation et de votre utilisation de ce site web.

2. Données stockées et usage

Ce site est une application web statique s'exécutant entièrement côté client (dans votre navigateur). Il fonctionne selon les principes suivants :

  • Local Storage (Stockage Local) : Toutes les données que vous créez (ajout d'histoires personnalisées, programmation de dates de chapitres) sont stockées exclusivement dans l'espace de stockage local (localStorage) de votre propre navigateur internet.
  • Aucune transmission : Aucune de ces données n'est envoyée, transmise ou stockée sur un serveur distant externe. Nous n'avons aucun moyen d'accéder aux histoires ou programmations que vous ajoutez sur votre poste.
  • Aucun cookie publicitaire : Ce site n'utilise pas de traceurs tiers ni de cookies publicitaires pour collecter des données à des fins de ciblage.

3. Liens de Redirection Externe

Ce site propose des liens vers la plateforme officielle Wattpad afin que vous puissiez y lire les histoires correspondantes. Une fois que vous quittez ce site pour Wattpad, le traitement de vos données personnelles est soumis à la politique de confidentialité propre de Wattpad (que nous vous invitons à consulter).

4. Contrôle de vos données

Puisque les données sont stockées uniquement sur votre appareil, vous disposez d'un contrôle total sur celles-ci :

  • Vous pouvez supprimer les histoires et parutions une par une depuis l'interface du panneau "Gérer".
  • Vous pouvez effacer l'intégralité des données à tout moment en vidant les cookies et données de site (ou le stockage local) de votre navigateur internet.

5. Contact

Pour toute question concernant l'utilisation du site, n'hésitez pas à vous référer à nos comptes officiels ou profil Wattpad mentionnés sur la page À Propos.
